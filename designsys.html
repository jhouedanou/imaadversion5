Notes importantes pour le graphiste

  • 📸 Photos équipe : Remplacer les placeholders (initiales) par des portraits professionnels en noir & blanc avec filtre bleu nuit. Format carré 600×600px minimum.
  • 🖼️ Photos projets : Remplacer les fonds dégradés des cartes projets par des photos réelles des infrastructures concernées. Ratio 16:9, hauteur 200px.
  • 🗺️ Carte Afrique : L'SVG simplifié de la section Impact peut être remplacé par une carte Mapbox/Leaflet interactive avec les pins de localisation réels.
  • 🎥 Hero vidéo : Le fond hero peut accueillir une vidéo courte (5–10s, loop, muted) en overlay sur le dégradé navy. Format MP4, taille < 5MB.
  • 📐 Logo : Fournir le fichier SVG final du logo IMAAD pour remplacer le logo géométrique placeholder utilisé dans le code.
  • 📄 Brochure : Ajouter le lien vers le PDF de la brochure institutionnelle dans le footer (section Ressources).
  • 🔗 LinkedIn : Mettre à jour les liens LinkedIn des membres de l'équipe dans equipe.html.
  • ⚡ Performance : Compresser toutes les images (WebP recommandé), lazy-load les images hors viewport. Objectif : Lighthouse Score ≥ 90.
  • ♿ Accessibilité : Le design respecte WCAG AA (contraste navy/#fff et gold/#0B1F3A validés). Ajouter les attributs alt sur toutes les images.
